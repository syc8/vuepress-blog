### React核心价值: 组件化 & 数据驱动视图
#### 数据驱动视图

> 类比公式  UI = f(state)
> f: 代表函数, 在react中也代表组件
- 定义好数据和UI的显示规则, 即UI = f(state)
- 只关注业务数据的修改, 不用再操作DOM, 增加开发效率
- 尤其对于DOM结构复杂的大型项目
